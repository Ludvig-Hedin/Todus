# Archived Apps

This folder contains legacy/duplicate app implementations kept for reference only.

Do not use these for active development or release:

- `native`
- `webview-swift`
- `apple`

Active app targets are:

- `apps/ios` (iPhone native)
- `apps/macos` (desktop webview wrapper)
- `apps/mail` (web)
- `apps/server` (backend)
